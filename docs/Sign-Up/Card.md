# Card Layout Component

## Props 

| Prop Name       | type       | required | values                                    |
|:----------------|:-----------|:---------|:------------------------------------------|
| title           | `string`   |          | default: `blank`                          |



## What They Do

`title` - Sets a Title at the top of the card <br>
`children`- Displays all children components wrapped inside the card <br>
